# 마이크로그린 대기업급 설치 계획

## 생산 셀

- 기본 단위: 6단 수직 재배랙 1대
- 랙당 트레이: 48장
- 권장 시작 규모: 6랙, 총 288트레이
- 작물 순환: 7-10일 수확 작물 기준 매일 파종/수확

## 구역

| 구역 | 목적 | 주요 장비 |
| --- | --- | --- |
| 발아실 | 암발아, 압착, 초기 균일화 | 선반, 덮개, 웨이트, 온습도 센서 |
| 생장실 | 광합성, 색, 수율 관리 | 수직랙, LED, 팬, CO2, PPFD 센서 |
| 급수실 | 물 저장, pH/EC 관리 | 저수조, 펌프, 필터, pH/EC, 수위 센서 |
| 수확실 | 절단, 선별, 계량 | 위생 작업대, 저울, QC 기록 |
| 포장실 | 포장, 라벨, 출하 | 실링기, 라벨러, 냉장 보관 |

## 센서

- 온도/습도: 구역별 1개 이상
- CO2: 생장실 1개 이상
- PPFD: 랙 높이별 이동 측정 또는 고정 센서
- 수위: 저수조별 1개
- pH/EC: 저수조별 1세트
- 누수 감지: 급수 라인 하부

## 제어

- 조명: 릴레이 또는 0-10V 디밍
- 팬: PWM 또는 단계 제어
- 펌프: 릴레이 + 수동 바이패스
- 경보: 부저, 문자/앱 알림, 대시보드 알림

## 표준 환경

| 항목 | 목표 |
| --- | --- |
| 온도 | 20-24°C |
| 습도 | 45-60%, 발아실은 단기 70% 허용 |
| CO2 | 400-900ppm |
| PPFD | 150-300 µmol/m²/s |
| pH | 5.8-6.5 |
| 공기 흐름 | 작물 표면이 살짝 흔들리는 정도 |

## 위생

- 트레이는 사용 후 세척, 살균, 완전 건조합니다.
- 씨앗 로트, 파종일, 수확일, 작업자를 배치별 기록합니다.
- 재배 구역과 포장 구역을 분리합니다.
- 수확 도구는 배치 전환마다 소독합니다.
